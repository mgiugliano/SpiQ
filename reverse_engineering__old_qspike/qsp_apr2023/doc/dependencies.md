**QSpikeTools 2.0** is programmed in **Julia 1.3.0** and uses **bash** scripts for Linux. *INSTALL.sh* will download and install all the necessary Julia dependencies in their correct version.

## Julia packages needed:
- Conda 1.3.0
- DSP 0.6.2
- Dierckx 0.4.1
- FileIO 1.1.0
- HDF5 0.12.5
- JLD2 0.1.8
- Plots 0.28.1
- PyCall 1.91.2
- PyPlot 2.8.2
- StatsBase 0.32.0

For more details, please check the *Project.toml* and *Manifest.toml* files.
